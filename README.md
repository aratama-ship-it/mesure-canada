# MESURE — Canada

モントリオール、ガティノー、トロント、GTA、オタワを起点に、国外公募とカナダの文化芸術助成・有給プログラムを結び付ける検証用MVPです。フランス語を主表示とし、英語・日本語へ切り替えられます。

- 公開サイト: https://aratama-ship-it.github.io/mesure-canada/
- このリポジトリには現時点でオープンソースライセンスを設定していません。

## 今回検証する体験

1. 現在の居住都市、申請主体、分野を選ぶ
2. 在留資格、州・市の居住期間、SIN、到着時期など、表示された重要質問へ答える
3. 実在する国外公募を一つ選ぶ
4. 公募へ直接関連する制度と、居住地から探せるその他の制度を分けて確認する

単純な一覧ではなく、`Opportunity ↔ FundingMatch` を中心にしています。判定は「可能」「条件付き」「要確認」「対象外」の4状態で、在留資格の公式要項へのリンクと確認日もデータに保持します。

## データ

- `data/opportunities.json` — 国・都市を絶対値で持つ公募データ
- `data/funding.json` — カナダ、ケベック州、オンタリオ州、各都市の制度データ
- 2026年7月19〜20日に公式ページを確認した小規模サンプル

在留資格は、市民、永住者、Protected Person、永住申請中、就労可能な一時滞在、就労不可の一時滞在に分けています。公式要項が在留資格を明記しない場合は、自動的に利用可とせず「要確認」にします。
