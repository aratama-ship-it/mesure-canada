# MESURE — Québec

モントリオールを起点に、国外公募とケベック／カナダの文化芸術助成を結び付ける検証用MVPです。フランス語を主表示とし、英語・日本語へ切り替えられます。

## 今回検証する体験

1. 申請主体・居住地・分野を選ぶ
2. 在留資格、ケベック居住期間、必要に応じてコレクティブ構成または団体登録状況を選ぶ
3. 実在する国外公募を一つ選ぶ
4. 利用可能性のある助成と、招待状・契約等の未充足条件を確認する

単純な一覧ではなく、`Opportunity ↔ FundingMatch` を中心にしています。判定は「可能」「条件付き」「要確認」「対象外」の4状態で、在留資格の公式要項へのリンクと確認日もデータに保持します。

## データ

- `data/opportunities.json` — 国・都市を絶対値で持つ公募データ
- `data/funding.json` — ケベック／カナダの助成データ
- 2026年7月19〜20日に公式ページを確認した小規模サンプル
