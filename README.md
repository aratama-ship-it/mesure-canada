# MESURE — Canada

モントリオール、ケベック・シティ、ガティノー、トロント、GTA、オタワを起点に、国外公募とカナダの文化芸術助成・有給プログラムを結び付ける検証用MVPです。フランス語を主表示とし、英語・日本語へ切り替えられます。

- 公開サイト: https://aratama-ship-it.github.io/mesure-canada/
- このリポジトリには現時点でオープンソースライセンスを設定していません。

## 今回検証する体験

1. 現在の居住都市、申請主体、分野を選ぶ
2. 在留資格、州・市の居住期間、SIN、到着時期など、表示された重要質問へ答える
3. 実在する国外公募を一つ選ぶ
4. 公募へ直接関連する制度と、居住地から探せるその他の制度を分けて確認する

単純な一覧ではなく、`Opportunity ↔ FundingMatch` を中心にしています。判定は「可能」「条件付き」「要確認」「対象外」の4状態で、在留資格の公式要項へのリンクと確認日もデータに保持します。

## データ

- `data/opportunities.json` — 国・都市を絶対値で持つ公募データ
- `data/festival-radar.json` — 世界のサーカス／大道芸／フリンジ／映画祭と、カナダのプロ向けショーケース／市場を、募集中・開始予定・次回監視に分けた公式ソース台帳（2026年7月21日時点で95件）。CAFF加盟Fringeの未収録地域、MICC、En Piste、TOHU、Valspecのサーカス向けピッチ／初回ツアー／レジデンスも含む
- `data/funding.json` — カナダ、ケベック州、オンタリオ州、各都市の助成・有給支援制度21件
- 2026年7月19〜21日に公式ページを確認した段階的拡張中のデータセット

在留資格は、市民、永住者、Protected Person、永住申請中、就労可能な一時滞在、就労不可の一時滞在に分けています。公式要項が在留資格を明記しない場合は、自動的に利用可とせず「要確認」にします。

## 世界フェスティバル・ショーケース レーダー

レーダーは「世界中を一度で完全に列挙した」とは主張しません。国際フェスとショーケースの募集は毎年変わるため、公式URL、確認日、次回確認日、国際応募可か／地域・言語条件があるかを一緒に保存します。単体フェスだけでなく、CAFFやBNFNのように複数都市へつながる巡回回路、MICC・Contact ontarois・CINARS・RIDEAU・ROSEQのようなプロ向け市場、En Piste・TOHU・Valspecの創作・ツアー支援ルートも別レコードとして記録します。ショーケースは通常のフェス公募と混同せず、独立フィルターで確認できます。さらに「地域フェスティバル・出演者公募」「イベント出演」「振付家育成プログラム」「AIR・滞在制作」の募集形式タグで絞り込めます。タグは既存の名称・分類からだけ判定し、応募資格を推測しません。締切が過ぎたレコードは自動的に「次回を監視」へ変わり、終了済みの情報を募集中として表示しません。
